## `인접행렬` 2번째 문제 (실패 - 코테 끝나고 20분 후 성공)
- 0 (길)
- 1 (사람)
- -1 (장애물)
- 몇명의 사람을 태울 수 있는지

```java
import java.util.*;


class Main {
    public static void main(String[] args) {
        List<List<Integer>> list = new ArrayList<>();
        List<Integer> num1 = new ArrayList<>();
        num1.add(0);
        num1.add(1);
        num1.add(-1);
        list.add(num1);
        num1 = new ArrayList<>();
        num1.add(1);
        num1.add(0);
        num1.add(-1);
        list.add(num1);
        num1 = new ArrayList<>();
        num1.add(1);
        num1.add(1);
        num1.add(1);
        list.add(num1);
        System.out.println(collectMax(list));
    }
    static int[] firstX = new int[]{-1 , 0 , 1 , 0};
    static int[] firstY = new int[]{0 , 1 , 0 , -1};
    static int[][] memoization;
    static int totalRiders = 0;
    static int sX , sY;

    public static int collectMax(List<List<Integer>> mat) {
        sX = mat.size();
        sY = mat.get(0).size();
        if(mat.get(sX - 1).get( sY - 1) == -1) return 0;
        memoization = new int[sX][sY];
        recursive(mat , 0 , 0 );
        return totalRiders;
    }
    public static void recursive(List<List<Integer>> list , int x , int y){
        System.out.println(x + " " + y);
        if(memoization[x][y] != 1){
            if(list.get(x).get(y) == 1){
                totalRiders++;
                list.get(x).set(y , 0);
                memoization[x][y] = 1;
            }
            for(int i = 0 ; i < 4 ; i++){
                int moveX = x + firstX[i];
                int moveY = y + firstY[i];
                if(moveX >= 0 && moveY >= 0 && moveX < sX && moveY < sY && list.get(moveX).get(moveY) != -1){
                    recursive(list , moveX , moveY);
                }
            }
        }
    }
}
```

## `문자열` 3번째 문제 (통과)
- 일 월 년 문자열 리스트가 들어오면 문자열 변환
- 20th Oct 2052 ➜ `2052-10-20`

```java
import java.util.*;

class t20210331 {
    public static void main(String[] args) {
        List<String> dates = new ArrayList<>();
        dates.add("20th Oct 2052");
        dates.add("6th Jun 1933");
        dates.add("26th May 1960");
        dates.add("20th Sep 1958");
        dates.add("16th Mar 2068");
        dates.add("25th May 1912");
        dates.add("16th Dec 2018");
        for(String date : preprocessDate(dates)){
            System.out.println(date);
        }
    }

    public static List<String> preprocessDate(List<String> dates) {
        List<String> resultList = new ArrayList<>();
        dates.forEach(date -> {
            StringJoiner sj = new StringJoiner("-");
            String[] dateSplit = date.split(" ");
            sj.add(dateSplit[2]);
            sj.add(getMonth(dateSplit[1]));
            String dateStr = "0" + dateSplit[0].substring( 0 , dateSplit[0].length() - 2);
            sj.add(dateStr.substring(dateStr.length() - 2 , dateStr.length()));
            resultList.add(sj.toString());
        });
        return resultList;
    }

    public static String getMonth(String monthStr){
        String month = "";
        if("Jan".equals(monthStr)) month = "01";
        else if("Feb".equals(monthStr)) month = "02";
        else if("Mar".equals(monthStr)) month = "03";
        else if("Apr".equals(monthStr)) month = "04";
        else if("May".equals(monthStr)) month = "05";
        else if("Jun".equals(monthStr)) month = "06";
        else if("Jul".equals(monthStr)) month = "07";
        else if("Aug".equals(monthStr)) month = "08";
        else if("Sep".equals(monthStr)) month = "09";
        else if("Oct".equals(monthStr)) month = "10";
        else if("Nov".equals(monthStr)) month = "11";
        else if("Dec".equals(monthStr)) month = "12";
        return month;
    }
}
```
