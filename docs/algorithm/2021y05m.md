## [5월 문제 해설](https://prgms.tistory.com/57)

```java
class Solution {
    public int solution(int left, int right) {
        int answer = 0;
        int valueCnt;
        for(int i = left ; i <= right ; i++){
            valueCnt = 0;
            for(int j = 1 ; j <= i ; j++){
                if(i % j == 0){
                    valueCnt++;
                }
            }
            if(valueCnt % 2 == 0){
                answer += i;
            }
            else answer -= i;
        }
        return answer;
    }
}
```
