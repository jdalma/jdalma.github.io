# **var** vs **val**

**var**     
가변 프로퍼티  
코틀린 컴파일러가 비공개 필드, 게터,세터를 생성한다

**val**     
불변 프로퍼티   
코틀린 컴파일러가 비공개 필드, 게터를 생성한다
